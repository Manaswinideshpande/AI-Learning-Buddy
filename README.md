# AI-Learning-Buddy
AI Learning Buddy is an AI agent that takes user input and provides explanation of user selected topic along with that it provides real world examples to make understand the topic to user also it creates a quiz on user selected topic.
# AI Learning Buddy

An AI-powered educational assistant built using:

- Streamlit
- Google Gemini API
- Prompt Engineering

Features

- Beginner Mode
- Intermediate Mode
- Advanced Mode
- Real-Life Examples
- Quiz Generator
- Ask Anything

Built as part of the EmpowerHer AI Program.
